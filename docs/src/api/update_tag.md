# 更新用户个人TAG

**必选参数** : 无

**接口地址** : `/update/tag`

**可选参数** : `tags` : 要设置的TAG内容 删除公告留空或省去即可，各TAG长度小于10字符，最多5个TAG，各TAG之间用","(%2C)分隔,重复TAG无效

**调用例子** : `/update/tag`

**说明** : 无

**响应说明** :

根对象：

| 字段      | 类型  | 内容   | 备注                                                             |
|---------|-----|------|----------------------------------------------------------------|
| code    | num | 返回值  | 0：成功<br />-101：账号未登录<br />-111：csrf校验失败<br />-400：请求错误（超出长度限制） |
| message | str | 错误信息 | 默认为0                                                           |
| ttl     | num | 1    |                                                                |
