# 更新用户个人空间公告

**必选参数** : 无

**接口地址** : `/update/space/notice`

**可选参数** : `notice` : 要设置的公告内容 删除公告留空或省去即可，少于150字符

**调用例子** : `/update/space/notice`

**说明** : 无

**响应说明** :

根对象：

| 字段      | 类型  | 内容   | 备注                                                                           |
|---------|-----|------|------------------------------------------------------------------------------|
| code    | num | 返回值  | 0：成功<br />-101：账号未登录<br />-111：csrf校验失败<br />-304：未修改<br />-400：请求错误（超出长度限制） |
| message | str | 错误信息 | 默认为0                                                                         |
| ttl     | num | 1    | 作用尚不明确                                                                       |
