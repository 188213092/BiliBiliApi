# 申请二维码

**必选参数** : `无`

**接口地址** : `/login/apply/qrcode`

**可选参数** : `无`

**调用例子** : `/login/qrcode`

**说明** : 二维码超时时间 : `180秒`

**响应说明** :

根对象：

| 字段      | 类型  | 内容   | 备注   |
|---------|-----|------|------|
| code    | num | 返回值  | 0：成功 |
| message | str | 错误信息 |      |
| ttl     | num | 1    |      |
| data    | obj | 信息本体 |      |

`data` 对象：

| 字段         | 类型  | 内容               | 备注     |
|------------|-----|------------------|--------|
| url        | str | 二维码内容 (登录页面 url) |        |
| qrcode_key | str | 扫码登录秘钥           | 恒为32字符 |
