# 获取用户个人空间公告

**必选参数** : `mid` : 用户mid

**接口地址** : `/get/space/notice`

**可选参数** : `无`

**调用例子** : `/get/space/notice`

**说明** : 无

**响应说明** :

根对象：

| 字段      | 类型  | 内容   | 备注                  |
|---------|-----|------|---------------------|
| code    | num | 返回值  | 0：成功<br />-400：请求错误 |
| message | str | 错误信息 | 默认为0                |
| ttl     | num | 1    |                     |
| data    | str | 公告信息 | 无则为空                |
