# 获取用户最近点赞的视频明细

**必选参数** : `mid` : 用户mid

**接口地址** : `/get/like/video`

**可选参数** : `无`

**调用例子** : `/get/like/video`

**说明** : 无

**响应说明** :

根对象：

| 字段      | 类型                      | 内容   | 备注                                       |
|---------|-------------------------|------|------------------------------------------|
| code    | num                     | 返回值  | 0：成功<br />-400：请求错误<br />53013：用户隐私设置未公开 |
| message | str                     | 错误信息 | 默认为0                                     |
| ttl     | num                     | 1    |                                          |
| data    | 隐藏时：null<br />公开时：array | 信息本体 |                                          |

`data`数组：

| 项   | 类型  | 内容        | 备注  |
|-----|-----|-----------|-----|
| 0   | obj | 点赞视频1     |     |
| n   | obj | 点赞视频（n+1） |     |
| ……  | obj | ……        |     |
